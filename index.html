CSCI 488 ~ Senior Seminar in Computer Science ~

About This Page
This course focuses on Database Driven Web Applications and core Software Engineering principles.
This page intended to provide homework assignments, course materials, and other relevant information, rather than serving as a showcase for visual design, as this is not a web design course.
Instead, the primary emphasis is placed on server-side programming, database development, and building reliable, scalable applications that follow sound engineering practices.
